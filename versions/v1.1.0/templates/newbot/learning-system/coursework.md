## How This File Works

One entry per lesson or exercise unit. Newest at the top. Each entry carries enough to be useful months later without re-reading source material.

Entry template:

```
### {DATE} - {Module / topic} - {Lesson or exercise name}
- **Source:** {Chapter / video / problem-set reference}
- **What:** {One-line description of what the lesson covers}
- **Did:** {What was actually done - read, solved, built}
- **Result:** {Got it / partial / stuck - and what produced that outcome}
- **Artifact:** {Path to code, notes, or working file if any}
- **Next:** {Follow-up if any - re-read, redo, move on}
```

When an entry is fully closed (covered, no follow-up needed), it stays here as a record. Coursework files do not rotate to a LOG - they ARE the log.
